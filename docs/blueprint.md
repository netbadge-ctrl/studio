# **App Name**: IDC Ops Manager

## Core Features:

- Server Location: Display a server location, including module, rack, and U-position.
- Configuration Comparison: Show current vs target configuration to make differences visible.
- Real-time Component Management: Log component requests in real-time, showing their storage box location.
- Visual Operation Guides: Display mainboard/backplane slot diagrams and operation videos.
- Work Order Distribution: Manage work order distribution from the group leader dashboard to staff members. Adjustable assignments supported.
- Automated Location Suggestions: Use an AI tool that will, by reasoning about user-selected criteria (component type, stock location etc.) and the available options, auto-populate stock locations.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a reliable and professional feel, inspired by server room environments.
- Background color: Light gray (#ECEFF1), providing a neutral backdrop for detailed information.
- Accent color: A vibrant orange (#FF9800) to highlight critical actions and alerts.
- Font pairing: Use 'Inter' (sans-serif) for both headlines and body text due to its modern and neutral appearance, perfect for technical documentation.
- Code font: 'Source Code Pro' for any code snippets or SNs needing display in the app.
- Crisp, professional icons representing different server components and actions. Use color to indicate status.
- Subtle animations to guide users through complex tasks and highlight changes in real-time.
